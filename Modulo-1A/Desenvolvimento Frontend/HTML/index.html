Resumo da História do HTML

Resumo

O HTML(HyperText Markup Language) como próprio nome já diz é uma linguagem de marcação muito utilizado atualmente na construção de sites na Web.

Ela foi criada em 1991 por Tim Berners-Lee e sendo padronizada ao longo do tempo. Atualmente o HTML funciona na web como se fosse o esqueleto da página acompanhada do CSS que dá forma e estiliza esse esqueleto e do JavaScript que tem o papel de tornar esse esqueleto dinâmico e interativo.
